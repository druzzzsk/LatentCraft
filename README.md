# LatentCraft

**LatentCraft** — проект по оптимизации молекулярных свойств через навигацию в латентном пространстве молекулярных VAE.

Идея: структуры (SMILES / графы) кодируются VAE в непрерывное латентное пространство; в нём выполняется оптимизация целевых свойств (logP, QED, SA), после чего результат декодируется обратно в молекулу. Используются проверенные open-source VAE; методы оптимизации (градиентный подъём, Bayesian optimization, CMA-ES) сравниваются в едином evaluation framework.

Подробный план этапов — в [docs/ROADMAP.md](docs/ROADMAP.md).

## Структура репозитория

| Каталог | Назначение |
|---------|------------|
| `configs/` | YAML-конфиги экспериментов |
| `data/` | Данные и скрипты загрузки / предобработки |
| `models/` | Обёртки над внешними VAE |
| `optimizers/` | Реализации методов оптимизации |
| `evaluation/` | Метрики, scoring, визуализация |
| `experiments/` | Скрипты запуска экспериментов |
| `notebooks/` | EDA и анализ результатов |
| `results/` | Логи, чекпоинты, артефакты (не коммитятся) |
| `docs/` | Документация |

## Установка

```bash
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -e ".[dev]"
```

Зависимости для химии (например RDKit) и обучения моделей добавляются по мере этапов roadmap; см. `pyproject.toml`.
